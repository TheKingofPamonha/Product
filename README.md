# Product Preview Card Component - Frontend Mentor 
Esta é a minha solução para o desafio [Social links profile challenge do Frontend Mentor](https://www.frontendmentor.io/challenges/product-preview-card-component-GO7UmttRfa)
## 🔎 Visão Geral

### O Desafio

Os usuários devem ser capazes de:
- Visualizar o layout ideal para o componente dependendo do tamanho da tela do seu dispositivo (Responsividade).
- Ver os estados de *hover* (passar o mouse) e foco para os elementos interativos, como o botão de compra.
